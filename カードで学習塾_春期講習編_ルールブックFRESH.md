# **「ボードでRinkai 春期講習編」ルールブック**

## **1\. ゲームの概要**

本ゲームは、1〜6人のプレイヤーが学習塾の「室長」となり、春期シーズン（1月下旬〜5月上旬。半月を1ターンとする）の教室運営をシミュレーションするボードゲームです。8ターンの間に、授業を提供し、生徒を増やし、教室の評判を高め、ライバル教室よりも優れた成果を出すことを目指します。

* **プレイ人数:** 1〜6人  
* **プレイ時間:** 20分（目安）

## **2\. ゲームの目的（勝利条件）**

ゲームを進めることで得られる「体験」「入塾」「満足」「経理」の4指標を高めることで、8ターン終了時に、以下の3つの目標で獲得した「目標ポイント」の合計が最も高いプレイヤーが勝者となります。

* **退塾目標:** ゲーム終了時の退塾数(経理・満足の目標値(各15)に対する不足値の合計)で判定。  
  * **4以上:** **\-3ポイント**  
  * **2以下:** 0ポイント  
  * **1以下:** 1ポイント  
* **動員目標:** ゲーム終了時の (体験数 \+ 入塾数) の合計値で判定。  
  * **10以上 :** 1ポイント  
  * **12以上 :** 2ポイント  
* **入退目標:** ゲーム終了時の (入塾数 \- 退塾数) の合計値で判定。  
  * **8以上:** 3ポイント  
  * **10以上:** 4ポイント  
  * **12以上:** 5ポイント

退塾数 \=MAX \[15-経理,0\]+MAX\[15-満足,0\]  
※目標ポイントが同数の場合は、「入退差 (入塾数 \- 退塾数)」が多いプレイヤーの勝利となります。

## **4\. ゲームの準備**

1. 各プレイヤーはプレイヤーボードを受け取り、各ステータスを初期値にセットします。

| ステータス名 | 初期値 | 目標値 | 最小値 | 最大値 |
| :---- | :---- | :---- | :---- | :---- |
| 体験 | 0 | 10, 12 | 0 | なし |
| 入塾 | 0 | 8, 10, 12以上 | 0 | なし |
| 満足 | 3 | 15 | 0 | なし |
| 経理 | 3 | 15 | 0 | 15 |

2. 全員共通の**「基本カード」**(レアリティNのカード4種類×2枚ずつ)を受け取り、これを各プレイヤーの初期デッキ（個人の山札）とします。  
3. アクションカードをレアリティ毎に分け、それぞれシャッフルして研修会場の山札にします。  
4. レアリティがRのアクションカードの山から1人4枚カードを引き、うち2枚を自分のデッキに獲得します。選ばなかった2枚は研修会場の捨て札にします。  
5. 各プレイヤーは自分のデッキをシャッフルし、**手札を4枚引きます。**

## **5\. ゲームの流れ**

ゲームは全8ターンで構成されます。1ターンは「研修」「教室行動」「教室会議」の3つのフェーズで進行します。

### **【研修】 カード獲得**

全員が一斉に、ターン毎に定められたレアリティのカード（習得候補）を3枚引き、そのうち1枚を自分のデッキに加えます（習得）。  
残りはレアリティ別に、研修会場の捨て札置き場に置きます。研修会場の山札がなくなったら、研修会場の捨て札をシャッフルして補充します。  
例外的に、5月上旬に習得したカードは必ず5月上旬ターンの手札に入ります。

| 週 | 習得カードレアリティ |
| :---- | :---- |
| 1月下旬 | R |
| 2月上旬 | SR |
| 2月下旬 | R |
| 3月上旬 | SSR |
| 3月下旬 | SSR |
| 4月上旬 | SR |
| 4月下旬 | SR |
| 5月上旬 | SR |

### **【教室行動】 アクション実行**

1. 各プレイヤーは、このターンに実行するアクションカード（以下、カード）を手札から選び、対応するスタッフ（室長・専任講師・事務）の枠に配置します。原則として、1ターンにつき**各スタッフが1つのアクション**を実行できます（合計3アクション）。  
2. 全プレイヤーがカードを配置し終えたら、スタートプレイヤーから順番に、配置したカードの効果を解決します。3アクションの処理順はプレイヤーが選択します。  
   * 「体験」「入塾」「満足」「経理」のうち、カードで指定された指標のカウンターを進めます。  
   * 「入塾」の値を増やすには、同数以上の「体験」が必要です。(例: 体験8、入塾6の状況で、入塾+3の効果のカードを発動しても、入塾数は2だけ増加して、入塾8となります。)  
   * ターン毎に定められた「おすすめ行動」があります。指定されたカテゴリのアクションカードの枚数につき、対応するステータスが1上昇します。

| 週 | おすすめ行動カテゴリ | 対応ステータス |
| :---- | :---- | :---- |
| 1月下旬 | 動員 | 体験 |
| 2月上旬 | 応対 | 満足 |
| 2月下旬 | 動員 | 体験 |
| 3月上旬 | 庶務 | 経理 |
| 3月下旬 | 教務 | 入塾 |
| 4月上旬 | 応対 | 満足 |
| 4月下旬 | 教務 | 入塾 |
| 5月上旬 | 庶務 | 経理 |

### **【教室会議】 カード削除**

このフェーズは以下の順番で処理を行います。

1. **アクションカード削除**: このターンに使用したカード、残った手札、習得したカードを全て一旦デッキに戻します。デッキを見て、ターン毎に定められた枚数を上限に、任意のカードをゲームから取り除きます（削除）。

| 週 | 削除枚数 |
| :---- | :---- |
| 1月下旬 | 2 |
| 2月上旬 | 2 |
| 2月下旬 | 1 |
| 3月上旬 | 1 |
| 3月下旬 | 1 |
| 4月上旬 | 1 |
| 4月下旬 | 1 |
| 5月上旬 | 0 |

2. **手札補充:** 自分のデッキをシャッフルし、新たに4枚カードを引きます。

### **【ゲーム終了時処理】**

**集約〆切 (8ターン目終了時):**  
目標ポイントを計算し、勝者を決定します。

## **7\. アクションカード**

アクションカードには、以下の情報が書かれています。

1. カード名  
   * カードサムネイルに記載されます。  
2. レアリティ（N(初期入手)・R・SR・SSR）  
   * 高いほど、効果やコストがより大きいことを表します。  
     ゲーム後半には、高いレアリティのカードが手に入りやすくなります。  
   * カードサムネイルに記載されます。  
3. カテゴリ（動員・教務・応対・庶務）  
   * 主に「外部動員」「授業実施」「内部満足」「経理処理」のいずれに関連するかを表します。  
4. 各スタッフが実行した場合の効果量  
   * 「+X」といった効果詳細文のXの値が、スタッフ別に記載されています。  
   * 「NG」と書かれている職種のスタッフにカードを配置した場合、そのアクションは実行されません。  
5. 効果概要(topEffect)  
   * 「体験2」などのように、どの値がいくつ変化するかが記載されています。  
   * カードサムネイルに記載されます。  
   * 記号の意味は以下の通りです。

| 記号 | 意味 |
| :---- | :---- |
| 室 | 室長が実行した場合に限る効果がある |
| 講 | 講師が実行した場合に限る効果がある |
| 事 | 事務が実行した場合に限る効果がある |
| \* | 条件を満たしている場合のみ発動 |
| \! | マイナス効果がある |

6. 効果詳細(effect)  
   * カードの具体的な効果が書かれています。ルールブックと衝突する記述があれば、カードの効果詳細が優先されます。  
   * 記号の意味は以下の通りです。  
     

| 記号 | 意味 |
| :---- | :---- |
| 【】 | カッコ内に記載された職種のスタッフにだけ配置することができる。 |
| 〈〉 | 以降の効果は、カッコ内に記載された条件を満たしている場合のみ発揮される。 |

## **7\. スタッフ**

各プレイヤーの教室には「室長」「専任講師」「事務」の3名のスタッフがいます。1ターンにつき原則1枚のアクションカードを配置することで、カードの効果が発揮されます。  
※初期バージョンのカードプールでは、1スタッフに対して配置できるカードは1枚で、それ以上配置することは禁止される（エラーで拒絶される）仕様ですが、今後「他のカードが配置されているスタッフにも配置できる」という効果を持つカードが追加される予定です。
