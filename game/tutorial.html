ボードでRinkai ルール説明

学習塾経営シミュレーション

あなたは学習塾の教室長。

講師・事務スタッフと一緒に、体験生の勧誘・授業・満足度向上・経理手続きをバランスよく進めて、春期講習を大成功させましょう！

ゲームの流れ（全8ターン）

  1. [研修] カードを選択肢から1枚ゲット
  2. [行動] 手札4枚中3枚をスタッフ3人に配置
    • 使ったカードに応じて教室パラメータがUP！
  3. [会議] デッキから不要なカードを選んで削除

1月下旬〜5月上旬までの8ターン（各ターン0.5ヶ月）を繰り返します。

4つの教室パラメータ

カードは「動員」「教務」「応対」「庶務」の4区分あり、それぞれ対応するパラメータが向上します。

【体験】 (動員)
体験授業の申込件数。まずはこれを集めないと始まりません！
【入塾】 (教務)
新規入塾者数。体験生がいないと上がりません！
目標は8, 10, 12, 15名の4段階。
【満足】 (応対)
既存生徒の満足度。
終了時に15を下回った数だけ生徒が退塾してしまいます！
【経理】 (庶務)
授業料や契約書類の手続き進捗度。
終了時に15を下回った数だけ生徒が退塾してしまいます！

カードの見方

  • カード上段: カード名、カテゴリ、レアリティ
  • カード下段: 効果テキスト (長押しで詳細確認可能)

カードには配置条件や発動条件があるものも存在します。次のページで詳しく見てみましょう。

配置条件と発動条件

【 】: 配置条件

カードを配置できる職種（室長・講師・事務）が限定されているカードです。

例：『備品注文』は【事務】にしか配置できません。（経理+2, 満足+1）

〈 〉: 条件付き効果

特定の職種に配置した時だけ効果が発揮（または変化）します。

例：『休み時間トーク』
室長に配置 → 満足+1
〈講師・事務〉に配置 → 満足+3（話が盛り上がる！）

営業目標：退塾を防げ！

【退塾目標】

「満足」と「経理」不足による退塾者数を規定以下に抑えるのが最重要課題です。

  • 目標値は 2名, 1名, 0名 の3段階。
  • 退塾が4名以上だと評価大幅減点！

⚠️ 注意！

片方のパラメータだけ高くてもダメです。
「満足」「経理」両方が15に達しないと退塾者が出ます。バランス良く上げましょう。

営業目標：生徒を増やせ！

【動員目標】

体験申込数を競います。目標は 10, 12, 15名の3段階。

【入塾・退塾差目標】

春期終了時の生徒純増数 (入塾数) - (退塾数) を競います。

  • 目標は 8, 10, 12, 15名の4段階。

つまり…

入塾させるには「体験」が必要。
退塾を防ぐには「満足」と「経理」が必要。
結局4項目とも必要です！

スコア基準: FRESH

FRESHのスコアは、3つの観点別得点（退塾・動員・入退差）の合計で決まります。

退塾目標（満足・経理で決まる退塾数）

  • (満足が15未満の不足分) + (経理が15未満の不足分) = 退塾数
  • 退塾 4名以上 → -3点（大幅ペナルティ）
  • 退塾 1名以下 → +1点

動員目標（体験数）

  • 体験 12以上 → +2点
  • 体験 10以上 → +1点

入退差目標（入塾数 - 退塾数）

  • 入退差 12以上 → +5点
  • 入退差 10以上 → +4点
  • 入退差 8以上 → +3点

S+精度スコア（観点別得点が満点8点のとき）

  • 観点別得点の合計が 8点 の場合、次の式で加点を計算します。
  • 8 + 0.5×(min{体験,30}-12)÷18 + 1.5×(min{入退差,30}-12)÷18
  • 加点は 体験12以上・入退差12以上 から発生し、各値は 30 で上限です（最大10.0）。
  • 合計スコアが 9点以上 でS+ランクになります。

スコア基準: PRO

PROのスコアは、4カテゴリのポイント合計（最大16点）で決まります。

4つのカテゴリ（各最大4点）

  1. 動員ポイント — 体験数が多いほど高ポイント
  2. 退塾ポイント — 退塾数が少ないほど高ポイント（満足・経理の確保が重要）
  3. 入退差ポイント — 入塾数と退塾数の差が大きいほど高ポイント
  4. 満足ポイント — 満足値が高いほど高ポイント

評価の考え方

  • スコア合計で F〜SSの11段階評価
  • 満足を高めることが直接スコアに貢献（FRESHとの違い）
  • 満足・経理の両立で退塾を減らしつつ、満足ポイントも稼げる
